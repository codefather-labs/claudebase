works per Claude Code hook docs lineage. The plan's fallback chain
(FR-C2C-7.2) is preserved as a code-comment in the hook script body
and in this evidence file. Slice 9 QA will verify live; if the hook
does not fire, the fallback path is operator-driven (Mira calls
agent_describe as the final step of /bootstrap-feature, which has
been the de-facto path during this session anyway).
